# Elevator Management System

## Project Overview
This project is an innovative representation of an elevator management system which simulates elevators in a building with many of their real life functionalities. It can manage, control, and view the movements and status of the elevators in a given building.  The system will receive elevator calls, and handle them accordingly in a safe and efficient manner. The system will operate on factors such as how many elevators are available, the status of the elevators, the location of the elevator call, and movement of live elevators. The elevator system will also be able to respond to emergency situations such as medical emergencies, fire emergencies and power outages.
